sso-check-provider
==================

Simple SSO check provider


Installation
------------

    $ component install fs-components/sso-check-provider

API
---

```js
var sso = require("sso-check-provider");

sso('my-cookie-name', {
  swf: "http://my-host.com/path/to/easyxdm.swf"
});
```
